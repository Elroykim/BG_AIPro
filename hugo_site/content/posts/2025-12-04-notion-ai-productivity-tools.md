한 달 동안 AI 생산성 도구를 5개 번갈아 가며 써봤다. Notion AI, Otter.ai, Gamma, Perplexity, Mem. 결론부터 말하면 하나로 모든 걸 해결하는 도구는 없고, 작업 유형에 따라 맞는 도구가 다르다. 나는 주로 기술 블로그 작성, 회의 정리, 리서치 이 세 가지를 많이 하는데, 그 기준에서 느낀 점을 정리한다.

## 1. Notion AI: 이미 Notion 쓰고 있다면 가장 편한 선택

### 핵심 기능

Notion AI는 기존 Notion 워크스페이스 안에 AI가 녹아든 형태다. 별도 앱을 오가지 않아도 되는 게 제일 큰 장점이다.

- 텍스트 생성 및 편집: 글 요약, 다시 쓰기, 톤 변경, 번역, 길이 조절
- Q&A: 워크스페이스에 있는 문서를 기반으로 질문에 답변
- 데이터베이스 자동 채우기: 프로퍼티를 AI가 자동으로 분류하거나 요약
- 커넥터 통합: Slack, Google Drive, GitHub 등 외부 서비스 데이터까지 검색
- 작성 보조: 빈 페이지에서 주제만 주면 초안 생성

### 써보니 좋았던 점

직접 한 달간 쓰면서 가장 편했던 건 "이미 내 문서가 Notion에 있다"는 점이다. 예를 들어, 프로젝트 회의록이 Notion에 쌓여있으면 "이번 분기 회의에서 나온 기술 부채 관련 내용 정리해줘"라고 물으면 여러 페이지를 알아서 뒤져서 정리해준다. 별도로 문서를 복붙할 필요가 없다.

한국어 지원도 꽤 쓸 만하다. 한국어로 질문하면 한국어로 답하고, 한국어 문서 요약 품질도 괜찮다. 다만 전문 기술 용어가 많은 문서에서는 가끔 어색한 번역투가 나온다.

### 아쉬운 점

- AI 응답이 때때로 느림 (복잡한 쿼리는 5~10초 대기)
- 무료 플랜에서는 AI 사용 횟수 제한이 빡빡함
- Q&A 정확도가 문서 구조에 크게 좌우됨 (체계적으로 정리된 문서일수록 잘 찾음)
- 이미 Notion을 안 쓰고 있다면 진입 장벽이 있음

### 가격

| 플랜 | 가격 | AI 포함 여부 |
|------|------|------------|
| Free | 무료 | AI 제한적 (횟수 제한) |
| Plus | $10/월/인 | AI 포함 |
| Business | $18/월/인 | AI 포함 |
| Enterprise | 별도 문의 | AI 포함 + 고급 관리 |

Notion AI 애드온은 별도 구매가 아니라 유료 플랜에 기본 포함되어 있다. 무료 플랜에서도 제한적으로 사용 가능하다.

## 2. Otter.ai: 회의 녹취록의 게임 체인저

### 핵심 기능

Otter.ai는 실시간 음성 인식(STT)과 회의 요약에 특화된 도구다. Zoom, Google Meet, Microsoft Teams와 연동해서 회의에 봇이 자동으로 들어가 녹음하고 텍스트로 변환해준다.

- 실시간 음성→텍스트 변환
- 화자 구분 (누가 말했는지 자동 구분)
- 회의 요약 자동 생성 (핵심 포인트, 액션 아이템 추출)
- Zoom/Meet/Teams 자동 참석 및 녹음
- 대화 내용에서 키워드 검색

### 써보니 좋았던 점

영어 회의 녹취는 정말 잘한다. 화자 구분도 꽤 정확하고, 회의가 끝나면 요약까지 자동으로 만들어준다. 팀 미팅이 많은 사람에게는 생산성 도구 중에서 ROI가 가장 높을 수 있다.

### 한국어 지원: 솔직히 아쉽다

가장 큰 문제가 한국어다. 요즘 한국어 인식이 조금 나아지긴 했지만, 영어 대비 정확도가 많이 떨어진다. 특히 기술 용어가 섞인 한국어 대화에서는 "쿠버네티스"를 "쿠버 네트 이스"로 인식하는 식의 오류가 빈번하다. 한국어 회의가 주된 사용 목적이라면 클로바노트 같은 국산 도구가 더 나을 수 있다.

### 가격

| 플랜 | 가격 | 월간 녹음 | 핵심 기능 |
|------|------|----------|----------|
| Basic | 무료 | 300분 | 기본 녹음 + 요약 |
| Pro | $16.99/월 | 1,200분 | 고급 요약, 화자 구분 강화 |
| Business | $30/월/인 | 6,000분 | 팀 관리, 관리자 기능 |
| Enterprise | 별도 문의 | 무제한 | 커스텀 배포, SSO |

## 3. Gamma: 프레젠테이션을 AI로 만들기

### 핵심 기능

Gamma는 프레젠테이션, 문서, 웹페이지를 AI로 생성하는 도구다. 주제만 입력하면 슬라이드 디자인까지 알아서 해준다.

- 텍스트 입력으로 프레젠테이션 자동 생성
- 기존 문서(PDF, 텍스트 등)를 슬라이드로 변환
- 다양한 템플릿과 테마 지원
- 이미지 자동 배치 및 AI 이미지 생성
- 웹 문서 형태로 공유 가능 (별도 뷰어 불필요)
- 분석 기능: 누가 어디까지 봤는지 추적

### 써보니 좋았던 점

급하게 발표 자료를 만들어야 할 때 진짜 유용하다. 나는 사내 기술 공유 세미나용 자료를 Gamma로 초안을 뽑고, 세부 내용만 수정하는 식으로 썼는데 기존 대비 시간이 1/3로 줄었다.

디자인 감각이 없어도 괜찮은 수준의 슬라이드가 나온다는 게 큰 장점이다. PowerPoint에서 도형 하나하나 맞추던 시절을 생각하면 격세지감이다.

한국어 입력도 잘 받아들인다. "AI 트렌드 2025: 개발자가 알아야 할 변화"라고 주제를 주면 한국어로 슬라이드를 만들어준다. 다만 기본 템플릿의 영문 표현이 간혹 섞여 나오는 경우가 있어서 최종 검수는 필요하다.

### 아쉬운 점

- 복잡한 차트나 데이터 시각화는 약함
- 세밀한 레이아웃 커스터마이징이 제한적
- 생성된 콘텐츠가 일반적인 내용에 치우치는 경향
- PowerPoint로 내보내기 시 일부 서식이 깨질 수 있음

### 가격

| 플랜 | 가격 | AI 크레딧 | 핵심 기능 |
|------|------|----------|----------|
| Free | 무료 | 400 크레딧 | 기본 생성, Gamma 워터마크 |
| Plus | $10/월 | 무제한 | 워터마크 제거, PDF 내보내기 |
| Pro | $20/월 | 무제한 | 커스텀 브랜딩, 분석, 우선 처리 |

## 4. Perplexity: 리서치 작업의 표준

### 핵심 기능

Perplexity는 AI 기반 검색 엔진이다. 검색 결과를 종합해서 출처와 함께 깔끔한 답변을 만들어준다. Google 검색 + ChatGPT를 합쳐놓은 느낌이다.

- 웹 검색 기반 실시간 정보 제공
- 출처 표시 (모든 답변에 참조 링크 포함)
- Pro Search: 복합 질문에 대해 심층 리서치
- 학술 논문 검색 모드
- Collections: 리서치 결과 정리 및 공유
- Focus 모드: 검색 범위 지정 (웹, 학술, 유튜브, Reddit 등)

### 써보니 좋았던 점

기술 리서치를 할 때 Perplexity가 없으면 이제 불편하다. 예를 들어 "Llama 3.2와 DeepSeek-R1 벤치마크 비교"를 검색하면, 최신 벤치마크 데이터를 여러 출처에서 가져와 정리해준다. 기존에는 Google에서 링크 10개 열어서 일일이 비교했는데, 그 과정이 한 번에 끝난다.

특히 Pro Search 기능이 좋다. 단순 검색이 아니라 "최근 6개월간 Rust 웹 프레임워크의 벤치마크 변화 추이를 정리해줘" 같은 복합 요청을 하면, 여러 검색을 자동으로 수행해서 종합 보고서 형태로 정리해준다.

### 한국어 지원

한국어 질문에 한국어로 잘 답한다. 한국어 웹 소스도 잘 가져온다. 다만 한국어 전문 정보(법률, 정부 정책 등)는 영문 소스 위주로 답하는 경향이 있어서, 한국 특화 정보를 찾을 때는 직접 확인이 필요하다.

### 아쉬운 점

- 무료 플랜의 Pro Search 횟수 제한 (하루 5회)
- 오래된 정보와 최신 정보가 섞여 나올 때가 있음
- 코드 관련 질문에서는 ChatGPT나 Claude보다 약함
- 한국 로컬 정보(맛집, 지역 정보 등)는 약함

AI 업무 자동화에 관심 있다면 [AI 자동화 생산성 가이드]({{< relref "posts/2025-12-07-ai-productivity-automation.md" >}})도 함께 읽어보면 도움이 된다.

### 가격

| 플랜 | 가격 | Pro Search | 핵심 기능 |
|------|------|----------|----------|
| Free | 무료 | 하루 5회 | 기본 검색, 출처 표시 |
| Pro | $20/월 | 무제한 | 모든 Pro 기능, 파일 업로드, API |
| Enterprise | $40/월/인 | 무제한 | SSO, 관리자 기능, 내부 데이터 검색 |

## 5. Mem: AI 기반 개인 지식 관리

### 핵심 기능

Mem은 AI가 내장된 노트 앱이다. 메모를 자유롭게 적으면 AI가 자동으로 분류하고, 관련 메모끼리 연결해주고, 필요할 때 검색으로 찾아준다.

- 자동 정리: 태그나 폴더 없이 AI가 자동 분류
- 관련 메모 연결: 비슷한 내용의 메모를 자동으로 연결
- Smart Search: 자연어로 메모 검색 (예: "지난달 클라이언트 미팅에서 나온 예산 이야기")
- Mem Chat: 자신의 메모를 기반으로 AI 대화
- 타임라인 뷰: 시간 순서대로 메모 탐색

### 써보니 좋았던 점

"일단 적고 나중에 정리하자" 스타일인 사람에게 잘 맞는다. 나도 폴더 만들고 태그 다는 게 귀찮아서 Notion에 메모가 뒤죽박죽이었는데, Mem은 그냥 막 적어도 나중에 AI가 찾아준다.

Mem Chat 기능도 흥미롭다. 내가 지금까지 적어둔 기술 메모를 기반으로 "내가 작년에 메모한 것 중에 FastAPI 관련 내용 정리해줘"라고 하면 관련 메모들을 모아서 요약해준다. 개인 지식 베이스를 만드는 느낌이다.

### 아쉬운 점

- Notion만큼의 구조화된 문서 관리는 어려움 (데이터베이스, 칸반 등 없음)
- 팀 협업 기능이 약함 (기본적으로 개인용)
- 한국어 자동 분류/연결 정확도가 영어보다 떨어짐
- 메모가 수백 개 이상 쌓여야 AI 분류의 진가가 발휘됨
- 가격 대비 기능이 Notion보다 제한적이라는 의견도 있음

### 가격

| 플랜 | 가격 | 핵심 기능 |
|------|------|----------|
| Free | 무료 | 기본 메모, 제한된 AI |
| Mem Plus | $14.99/월 | 무제한 AI, Smart Search, Mem Chat |
| Team | $19.99/월/인 | 팀 공유, 권한 관리 |

## 8. 각 도구별 꿀팁

### Notion AI 활용 팁

```
1. AI 블록 사용: / 키 누르고 "AI" 입력하면 AI 블록 삽입 가능
2. Q&A 정확도 높이기: 문서를 체계적으로 정리할수록 AI 검색 정확도 올라감
3. 데이터베이스 자동 채우기: 프로퍼티 타입을 "AI Summary"로 설정하면
   각 행의 내용을 자동 요약
4. 번역: 다국어 문서를 통째로 선택하고 "번역" 기능 활용
```

### Perplexity 활용 팁

```
1. Focus 모드: 검색 범위를 "Academic"으로 설정하면 학술 논문 위주로 검색
2. Collections: 리서치 주제별로 컬렉션을 만들어 결과 누적 관리
3. 후속 질문: 답변 아래 제안되는 후속 질문을 활용하면 리서치 깊이가 올라감
4. 파일 업로드: PDF를 업로드하고 내용에 대해 질문 가능 (Pro)
```

### Gamma 활용 팁

```
1. 기존 문서 변환: 이미 작성된 Notion 문서나 Google Docs를 붙여넣기하면
   슬라이드로 변환 가능
2. 이미지 스타일: 생성 시 "일러스트", "사진", "미니멀" 등 이미지 스타일 지정
3. 발표자 노트: 각 슬라이드에 AI가 발표자 노트도 자동 생성
4. 웹 공유: 링크로 공유하면 별도 뷰어 없이 브라우저에서 바로 열림
```

## 끝으로

5개 도구를 전부 써보고 느낀 건, AI 생산성 도구는 결국 "내 워크플로우에 자연스럽게 녹아드느냐"가 핵심이라는 것이다. 아무리 좋은 도구도 기존 흐름에서 벗어나면 번거로워서 안 쓰게 된다. Notion을 이미 쓰고 있다면 Notion AI가 가장 자연스럽고, 영어 회의가 잦다면 Otter.ai의 효과가 즉각적이다. 하나만 골라야 한다면? 개인적으로는 Perplexity다. 리서치 시간이 줄어드는 게 체감이 가장 뚜렷했다.
