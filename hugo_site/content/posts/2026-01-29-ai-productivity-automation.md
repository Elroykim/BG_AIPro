## 1. 이메일 자동화

### 이메일 초안 작성

매일 수십 통의 이메일을 작성하는 대신, AI에게 초안을 맡기세요.

```
[프롬프트 템플릿]

다음 조건으로 비즈니스 이메일을 작성해줘:

받는 사람: [이름/직책]
목적: [회의 요청 / 프로젝트 업데이트 / 견적 요청 / 감사 인사]
핵심 내용: [전달할 주요 내용]
톤: [격식체 / 비즈니스 캐주얼 / 친근한]
길이: [짧게(3줄) / 보통(5~7줄) / 상세(10줄+)]
```

### Python으로 이메일 응답 자동화

```python
"""
이메일 자동 분류 및 초안 생성 스크립트
"""

import anthropic

client = anthropic.Anthropic()

def classify_and_draft(email_content: str) -> dict:
    """이메일을 분류하고 응답 초안을 생성합니다."""

    response = client.messages.create(
        model="claude-sonnet-4-20250514",
        max_tokens=1024,
        messages=[{
            "role": "user",
            "content": f"""다음 이메일을 분석하고 응답 초안을 작성해줘.

이메일 내용:
{email_content}

다음 형식으로 답변해:
1. 카테고리: [문의/요청/불만/정보공유/기타]
2. 긴급도: [높음/보통/낮음]
3. 핵심 요약: [1줄]
4. 응답 초안: [비즈니스 이메일 형식으로]"""
        }]
    )

    return response.content[0].text

# 사용 예시
email = """
안녕하세요, 김 팀장님.

다음 주 수요일 오후 2시에 프로젝트 킥오프 미팅을 진행하고 싶습니다.
참석 가능하신지 확인 부탁드립니다.
"""

result = classify_and_draft(email)
print(result)
```

## 3. 데이터 처리 자동화

### 엑셀/CSV 데이터 정리

```python
"""
AI 기반 데이터 정리 자동화
"""

import anthropic
import pandas as pd
import json

client = anthropic.Anthropic()

def clean_data_with_ai(df: pd.DataFrame, instructions: str) -> str:
    """AI에게 데이터 정리 코드를 생성하게 합니다."""

    sample = df.head(5).to_string()
    columns = list(df.columns)
    dtypes = df.dtypes.to_string()

    response = client.messages.create(
        model="claude-sonnet-4-20250514",
        max_tokens=2048,
        messages=[{
            "role": "user",
            "content": f"""다음 데이터를 정리하는 Python pandas 코드를 작성해줘.

컬럼: {columns}
데이터 타입: {dtypes}
샘플 데이터:
{sample}

정리 요구사항:
{instructions}

실행 가능한 Python 코드만 반환해. 설명은 필요 없어."""
        }]
    )

    return response.content[0].text

# 사용 예시
df = pd.read_csv("sales_data.csv")
code = clean_data_with_ai(df, """
1. 날짜 컬럼을 datetime으로 변환
2. 매출 금액에서 쉼표 제거하고 숫자로 변환
3. 빈 값은 0으로 채우기
4. 중복 행 제거
5. 날짜 기준 오름차순 정렬
""")
print(code)
```

## 4. 문서 작성 자동화

### 보고서 템플릿 자동 생성

```python
def generate_report(data: dict, report_type: str) -> str:
    """데이터 기반 보고서를 자동 생성합니다."""

    prompts = {
        "weekly": """주간 업무 보고서를 작성해줘.
포함 항목: 금주 실적, 주요 이슈, 차주 계획""",

        "monthly": """월간 성과 보고서를 작성해줘.
포함 항목: KPI 달성률, 주요 성과, 개선점, 다음 달 목표""",

        "project": """프로젝트 상황 보고서를 작성해줘.
포함 항목: 진행률, 완료 작업, 리스크, 다음 마일스톤""",
    }

    response = client.messages.create(
        model="claude-sonnet-4-20250514",
        max_tokens=2048,
        messages=[{
            "role": "user",
            "content": f"""{prompts[report_type]}

데이터:
{json.dumps(data, ensure_ascii=False, indent=2)}

형식: 마크다운
톤: 비즈니스 전문적
길이: A4 1페이지 분량"""
        }]
    )

    return response.content[0].text
```

## 6. 자동화 워크플로우 구축 (n8n / Zapier)

### 노코드 AI 자동화 예시

```
[워크플로우 1: 고객 문의 자동 처리]
이메일 수신 → AI 분류 → 담당자 자동 배정 → 초안 생성 → 슬랙 알림

[워크플로우 2: SNS 콘텐츠 자동 발행]
블로그 발행 → AI 요약 → SNS 포맷 변환 → 예약 발행 → 성과 추적

[워크플로우 3: 일일 리포트]
데이터 수집 (API) → AI 분석 → 리포트 생성 → 이메일 발송

[워크플로우 4: 경쟁사 모니터링]
웹 크롤링 → 변경 감지 → AI 요약 → 슬랙 알림
```

## 자동화 도입 3단계 전략

### 1단계: 빠른 승리 (Quick Wins)

```
투자: 0원~$20/월
시간 절감: 주 2~5시간

- ChatGPT/Claude로 이메일 초안 작성
- AI로 회의록 요약
- AI로 보고서 초안 생성
- 번역 도구 활용
```

### 2단계: 워크플로우 연결

```
투자: $20~50/월
시간 절감: 주 5~10시간

- n8n/Zapier로 반복 업무 자동화
- AI 코딩 도구로 개발 생산성 향상
- 데이터 처리 스크립트 자동화
- SNS 콘텐츠 반자동 생성
```

### 3단계: 시스템 구축

```
투자: $50~200/월
시간 절감: 주 10~20시간

- 커스텀 AI 챗봇 구축 (사내 지식 기반)
- API 연동 전자동 파이프라인
- 데이터 분석 대시보드 자동화
- AI 기반 고객 응대 시스템
```

## 자동화 도입 시 주의사항

### 1. 사람의 검증은 필수

```
AI 자동화의 원칙:

AI가 작성 → 사람이 검토 → 발행/전송

AI의 결과물을 무검증으로 사용하면 안 됩니다.
특히 외부 커뮤니케이션(이메일, 보고서)은
반드시 사람이 최종 확인하세요.
```

### 2. 민감한 정보 주의

```
AI 도구에 입력하면 안 되는 정보:
- 고객 개인정보 (이름, 연락처, 주민번호)
- 회사 기밀 (재무 정보, 전략 문서)
- 인증 정보 (API 키, 비밀번호)
- 법적 문서 (계약서 원문)

대안: 데이터를 익명화하거나 로컬 AI 모델 사용
```

### 3. 점진적 도입

```
❌ 모든 업무를 한 번에 자동화하려는 시도
✅ 가장 반복적인 업무 1개부터 시작

추천 순서:
1. 가장 시간이 많이 드는 반복 업무 파악
2. AI 도구로 30% 자동화
3. 결과 확인 후 범위 확대
4. 워크플로우 연결
5. 시스템화
```

## 마무리

AI 자동화는 "일자리를 빼앗는 기술"이 아니라 **"반복 작업에서 해방시켜 창의적인 일에 집중하게 하는 기술"**입니다. 하루에 이메일 작성 30분, 데이터 정리 1시간, 보고서 작성 1시간을 AI에게 맡기면, 그 시간을 기획, 전략, 창의적 문제 해결에 투자할 수 있습니다.

오늘 당장 할 수 있는 것부터 시작하세요. ChatGPT나 Claude에게 오늘 써야 할 이메일 초안을 요청해보는 것만으로도 AI 자동화의 가치를 체감할 수 있을 것입니다.
